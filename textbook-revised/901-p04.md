### Step 901.5: Register the Home Component in AppModule

The HomeComponent is declared directly in the root `AppModule` rather than using a separate feature module. This approach provides simpler setup for components that are always loaded.

In `src/app/app.module.ts`, ensure HomeComponent is imported and declared:

```typescript
// In src/app/app.module.ts
import { HomeComponent } from './features/home/home.component';

@NgModule({
  declarations: [
    AppComponent,
    HomeComponent,
    // ... other components
  ],
  imports: [
    BrowserModule,
    BrowserAnimationsModule,
    HttpClientModule,
    // ... other modules
  ],
  // ...
})
export class AppModule {}
```

**Module registration explained:**

| Property | Purpose |
|----------|---------|
| `declarations` | Components that belong to this module, including HomeComponent |
| Direct declaration | HomeComponent is eagerly loaded with the app, no lazy loading |
