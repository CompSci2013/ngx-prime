### Step 901.4: Create the Home Component Styles

Create `src/app/features/home/home.component.scss`:

```scss
// src/app/features/home/home.component.scss
// VERSION 1 (Section 901) - Home page styles with dark theme

.home-container {
  min-height: 100vh;
  background: linear-gradient(135deg, #1a1a1a 0%, #2d2d2d 100%);
  padding: 4rem 2rem;
  overflow-y: auto;

  /* Header Section */
  .header {
    text-align: center;
    margin-bottom: 4rem;
    animation: fadeInDown 0.8s ease-out;

    h1 {
      font-size: 3rem;
      font-weight: 700;
      color: #ffffff;
      margin: 0 0 1rem 0;
      letter-spacing: -0.5px;
      text-shadow: 0 2px 10px rgba(0, 0, 0, 0.3);
    }

    .subtitle {
      font-size: 1.2rem;
      color: #b0b0b0;
      margin: 0;
      font-weight: 300;
    }
  }

  /* Domains Section */
  .domains-section {
    margin-bottom: 4rem;
    animation: fadeInUp 0.8s ease-out 0.4s both;

    .domain-grid {
      display: grid;
      grid-template-columns: repeat(auto-fit, minmax(180px, 1fr));
      gap: 1.5rem;
      max-width: 1200px;
      margin: 0 auto;

      .domain-card {
        background: linear-gradient(135deg, rgba(255, 255, 255, 0.05) 0%, rgba(255, 255, 255, 0.02) 100%);
        border-radius: 12px;
        padding: 2rem 1.5rem;
        text-align: center;
        text-decoration: none;
        border: 1px solid rgba(255, 255, 255, 0.08);
        transition: all 0.3s ease;
        cursor: pointer;
        position: relative;
        overflow: hidden;

        &::before {
          content: '';
          position: absolute;
          top: 0;
          left: 0;
          right: 0;
          height: 3px;
          background: linear-gradient(90deg, #64c8ff, #4ba3d9);
          transform: scaleX(0);
          transition: transform 0.3s ease;
          transform-origin: left;
        }

        &:hover {
          transform: translateY(-6px);
          border-color: rgba(100, 200, 255, 0.2);
          background: linear-gradient(135deg, rgba(100, 200, 255, 0.1) 0%, rgba(75, 163, 217, 0.05) 100%);
          box-shadow: 0 12px 30px rgba(100, 200, 255, 0.15);

          &::before {
            transform: scaleX(1);
          }

          .card-icon {
            transform: scale(1.2);
          }
        }

        .card-icon {
          font-size: 2.5rem;
          display: block;
          margin-bottom: 1rem;
          transition: transform 0.3s ease;
        }

        h3 {
          font-size: 1.3rem;
          color: #ffffff;
          margin: 0 0 0.5rem 0;
          font-weight: 600;
        }

        p {
          color: #888;
          font-size: 0.9rem;
          margin: 0;
          line-height: 1.5;
        }

        &.disabled {
          opacity: 0.5;
          cursor: not-allowed;
          pointer-events: none;
        }
      }
    }
  }
}

/* Animations */
@keyframes fadeInDown {
  from {
    opacity: 0;
    transform: translateY(-30px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(30px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* Responsive Design */
@media (max-width: 768px) {
  .home-container {
    padding: 2rem 1rem;

    .header h1 {
      font-size: 2rem;
    }

    .domains-section .domain-grid {
      grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));
      gap: 1.5rem;
    }
  }
}
```

**CSS techniques used:**

| Technique | Purpose |
|-----------|---------|
| Dark gradient background | Creates modern, immersive feel with `linear-gradient(135deg, #1a1a1a, #2d2d2d)` |
| CSS Grid with `auto-fit` | Creates responsive columns that adapt to screen size |
| `translateY` on hover | Provides visual lift effect for interactive cards |
| `::before` pseudo-element | Animated top border that slides in on hover |
| Cyan accent colors | Uses `#64c8ff` / `#4ba3d9` for highlights and interactions |
| SCSS nesting | Keeps related styles organized and readable |
| Keyframe animations | Smooth entrance animations for header and domain grid |
