# 804: Statistics Panel Component

**Status:** Superseded by Section 808 (StatisticsPanel2Component)

---

## Notice

This section describes a `StatisticsPanelComponent` that was planned but not implemented. The functionality has been superseded by **StatisticsPanel2Component** documented in Section 808.

The directory `src/app/framework/components/statistics-panel/` exists as an empty placeholder in the codebase.

---

## Migration Guide

If you are looking for statistics panel functionality, please refer to:

- **Section 808: Statistics Panel 2 Component** - CDK mixed orientation chart grid with drag-drop reordering

The StatisticsPanel2Component provides:
- CDK horizontal drag-drop grid for chart arrangement
- Integration with BaseChartComponent for chart rendering
- Pop-out window support for charts
- Domain-agnostic configuration via DomainConfig.chartDataSources

---

## Original Learning Objectives (For Reference)

The planned component would have:
- Composed multiple chart components into a single panel
- Implemented drag-and-drop reordering with Angular CDK
- Coordinated chart clicks with URL state updates

These objectives are now fulfilled by StatisticsPanel2Component.
